# Projekt fragmentu budynku

  

## Opis projektu

Projekt wykonany w ramach studiów na kierunku Inżynieria Środowiska.

Celem projektu było opracowanie fragmentu budynku, wykonanie niezbędnych obliczeń oraz dokumentacji technicznicznej.

  

## Zakres projektu

- obliczenia projektowe;
w tym dobór odpowiednich materiałów oraz wymiarów elementów konstrukcji

- wykonanie rysunków technicznych w AutoCAD

  

## Wykorzystane narzędzia

- AutoCAD

- Excel

  
  

## Zakres opracowania

Projekt obejmuje:

- opis techniczny

- obliczenia

- rysunki
